# User Survey Application (Avalonia UI)

This is a cross-platform desktop application developed using **C#** and **Avalonia UI**. The application is a user survey form designed with a **Russian interface**, featuring real-time input validation and interactive UI elements.

## ðŸš€ Features

* **Cross-Platform:** Runs on Windows, Linux, and macOS.
* **Russian Interface:** Fully localized labels and messages.
* **Input Validation:**
    * **Phone Number:** Checks for non-numeric characters and displays an error message.
    * **Patronymic Logic:** A "No Patronymic" checkbox disables the input field if selected.
    * **Consent:** Submit button remains disabled until the user agrees to data processing terms.
* **Dynamic UI:** Real-time updates for sliders and input fields.
